## Что нужно знать при работе с fs_audit.sh
Файл запускается командой ./fs_audit.sh в командной строке, если предварительно зайти в домашнюю папку /os-hw03/
Файл fs_audit.sh выводит 4 секции:

# 1) Смонтированные файловые системы. 
За нее отвечает функция show_mounted_fs()

=== Смонтированные файловые системы ===
Filesystem     Type  Size  Used Avail Use% Mounted on
/dev/sdc       ext4  1007G  15G  941G   2% /
tmpfs          tmpfs  7.8G  ...

Вам нужно проверить куда записываются данные? Или продиагностировать проблемы с ФС?

# 2) Статистика директорий. 
Показывает количество файлов, директорий и связей. show_dir_stats()

# 3) Секция чтобы найти топ-5 файлов в директории. 
В данном случае это /etc. show_top_files()

# 4) Секция которая сравнивает df и du
находит разницу между состоянием файловой системы(свободном месте на ней) 
и использовании файлов
